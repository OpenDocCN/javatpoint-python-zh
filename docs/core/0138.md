## 什么是回文？

回文是即使数字和字母颠倒了也保持不变的数字或字母。

### 例如:

121，11，414，1221，74747 是回文数字。

妈妈，爸爸，女士，参考是回文字母。

JAVATPOINT、PROGRAM、JAVA 不是回文字母。

### 回文算法

*   读数字或字母。
*   将字母或数字保存在临时变量中。
*   颠倒字母或数字。
*   将临时变量与颠倒的字母或数字进行比较。
*   如果两个字母或数字相同，打印“这个字符串/数字是回文。”
*   否则打印，“这个字符串/数字不是回文。”

### 回文程序

**程序 1:回文串**

```

# This string is reverse.

   print("NOT PALINDROME !")
```
